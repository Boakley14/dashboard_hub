Tenant Metrics — Limited Data Available

Avg Tenure, Google Reviews, and Internal Reviews are not surfaced through the Lodestar REST API. These metrics require direct semantic model connectivity in Power BI Desktop or a custom API endpoint not yet configured.

Issue Management — Data Not Available

Work order and issue metrics (Active Issues, Avg Age, Units Unrentable, Units Offline) return null values from the Lodestar API. This data source requires a separate Work Orders connection to be configured.
