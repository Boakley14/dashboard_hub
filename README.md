# 10Fed Dashboard Hub

Central hub for 10 Federal dashboards, reporting tools, and data visualizations.

## Project Structure

```
10Fed-Dashboard-Hub/
├── src/          # Source code (HTML, JS, CSS, scripts)
├── assets/       # Images, logos, icons
├── data/         # Data files, CSVs, JSON feeds
├── docs/         # Documentation and notes
└── README.md
```

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/Boakley14/dashboard_hub.git
   ```
2. Open `src/` to find dashboard files.
